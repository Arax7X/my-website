FAQ

Questions, answered

Do I need to know how to code? +

Not really. Nova is a single HTML file with clear comments. Change the text, swap the colors at the top, and you're done. If you can edit a document, you can edit Nova.

Can I use it for paid client work? +

Yes. The Pro and Team licenses let you build unlimited sites, including paid projects for clients.

How do I change the colors? +

All brand colors live in one :root block at the top of the file. Edit a few variables and the entire page re-skins itself — including the dark and light themes.

Is it really just one file? +

Yes. HTML, CSS and a little JavaScript all live in landing.html. No build tools, no dependencies — just open it in a browser or drop it on any host.
