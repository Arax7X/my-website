Questions

The fine print, refined.

What exactly is Aurelia?+

A premium, multi-page website template kit. Six hand-built pages — landing, features, pricing, about, journal, and contact — that share one stylesheet and one small script, including the live 3D cosmic background. You get the full source. Open it in a browser and it just works.

Will the 3D background slow my site down?+

No. It's a single canvas drawn with a few KB of vanilla JavaScript — no video, no heavy library, no images to load. It pauses when the tab is hidden and stops entirely for visitors who prefer reduced motion, so it stays smooth and accessible.

Do I need to know how to code?+

Enough to edit text and swap a few colour tokens. Everything visual — down to the globe's aurora — is controlled by clearly-named CSS variables, so a full rebrand is a two-minute job. If you can edit a paragraph, you can ship Aurelia.

Can I use it for client work?+

Yes. The Studio tier covers unlimited client projects and lets you hand off the source; the Personal tier covers a single brand or product of your own. The one rule: don't resell Aurelia itself as a template.

How do I deploy it — and what if it's not for me?+

It's a plain folder of HTML, CSS, and JS — drag it onto Netlify, Vercel, GitHub Pages, or any static host and you're live. Every plan starts with a 14-day free trial, no card required, and if a paid month ever isn't worth it we'll refund it in full.
