What exactly do I receive?+

The complete Aurelia kit: six fully-designed HTML pages, one shared stylesheet, one shared script, and documentation. No build tools, no dependencies — open it in any browser and it works. Every subscription tier includes future updates.

Can I use Aurelia for client work?+

Yes — the Studio tier is built for exactly that. Use it across unlimited client projects and hand off the source. The Personal tier covers a single brand or product of your own.

Do I need to know how to code?+

Enough to edit text and swap a few colour tokens. Everything visual is controlled by clearly-named CSS variables, so rebranding is a two-minute job. If you can edit a paragraph, you can ship Aurelia.
